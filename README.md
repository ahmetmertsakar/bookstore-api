# 📚 Bookstore API

A simple RESTful API for managing a bookstore's inventory and orders.

---

## 🔧 Tech Stack

- **.NET 8** (ASP.NET Core Web API)
- **PostgreSQL**
- **EF Core**
- **Swagger** (for API documentation)
